HTML Validation Service

Welcome to the W3C HTML Validation Service. It checks HTML documents for conformance to W3C HTML and XHTML Recommendations and other HTML standards. Recent updates include:

  • Added support for XHTML 1.1, XHTML Basic 1.0, and MathML 2.0 (July 26, 2001)
  • Improved and expanded Character Encoding support(June 22, 2001)
  • Added support for ISO-HTML (October 18, 2000)
  • more...
