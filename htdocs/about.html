About the
W3C HTML Validation Service

The performance of this validator is a combination of the base system hardware and software and the network connectivity of the site which hosts it, providing very high validation performance to the Internet.

Hardware

The system consists of an Alpha Personal Workstation 500au with 128 megabytes of memory and a disk. This is a 500mhz Alpha 21164 processor with very fast main memory and cache. This is one step up from the low end Alpha workstation. If we need more speed, we can get it!

Software

The base system software is DIGITAL UNIX version 4.0C.

Network

The network which hosts this system is on a 100mbit/second one hop away from Compaq's Internet Exchange, in Palo Alto California. This provides as much bandwidth as we could possibly use at a site extremely well connected to many networks of the global Internet. The combination of fast hardware, a fast network, and great connectivity provides the service you see.
